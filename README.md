# Quick Search
Allows you to search for any script in the Unity project. And select the code editor


![sc](https://github.com/atoGames/Quick-Search/assets/57901158/8af9f8da-844d-4703-8d6c-5ed375d5f457)

# Features:
 + Search
 + Open
 + Select script
 + Select code editor

# Code editor:
  + VS Code
  + Visual studio
  + Rider

### Platforms:
- [x]  Windows

### Shortcut:
   `Ctrl + /`
